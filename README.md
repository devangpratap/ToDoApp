# ToDoApp (Java)

A simple **console-based To-Do List application** written in Java.  
Built to practice core Java concepts like classes, methods, loops, and user input.

---

## Features
- Add tasks
- View all tasks
- Remove tasks
- Mark tasks as completed
- Clear all tasks
- Menu-driven console interface

---

## Project Structure
ToDoApp/
├── ToDoApp.java    // Main application (UI + input handling)
├── ToDoList.java  // Task storage and logic
└── README.md

---

## How to Run

### Compile
```bash

javac ToDoApp.java ToDoList.java

java ToDoApp
